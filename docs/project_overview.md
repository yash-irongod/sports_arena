# Project Overview – Sports Arena

## Introduction
Sports Arena is a modern, responsive web portal designed for the sports department of K.R. Mangalam University. The website serves as a centralized platform where students can explore sports activities, register for events, book facilities, view records, and browse image galleries from previous tournaments.

The project was created as part of an academic assignment to demonstrate the use of Visual Studio Code, Git, GitHub, proper documentation, and organized repository structure.

---

## Objectives
- Build a functional and visually appealing sports portal.
- Practice version control using Git (meaningful commits, branching, pushing).
- Host the project using GitHub Pages.
- Maintain professional documentation and repository structure.
- Learn real-world development workflow.

---

## Features
- **Home Page:** Overview of sports activities and quick navigation.
- **Sports Page:** List of available sports and coaching information.
- **Events Page:** Upcoming fixtures and past results.
- **Registration Page:** Form for student and team registrations.
- **Facility Booking:** Book grounds, courts, and training areas.
- **Gallery:** Image gallery of past events.
- **Records Page:** Achievement highlights and scoreboards.
- **Contact Page:** University sports department details.

---

## Tech Stack
- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsiveness  
- **JavaScript (Vanilla JS)** – Interactivity (where used)  
- **VS Code** – Development environment  
- **Git & GitHub** – Version control + hosting
